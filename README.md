# gotfiles-sync

Keeps your configs in sync

![logo](media/logo.png)

## Features

- Tracks dotfiles & copies them into a dedicated repo.
- Shows concise TUI status for each managed app.

## Installation

```bash
git clone https://github.com/mordvn/dotfiles-sync.git
cd dorfiles-sync
```

## Usage

```bash
go run cmd/dotfiles-sync/main.go
```

Requirements:

- Go (latest stable)
- Git
